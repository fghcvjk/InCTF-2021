Description

Our IB team recovered a mobile phone from one of the members of the gang. Can you help out the team in finding answers for the following questions?

Questions:

    When did the Professor create the note for Rio?
        Answer in DD-MM-YYYY_HH:MM:SS. Timestamp in UTC

    Where did Professor and Rio, planned to meet for planning the heist?
        Format: Lat_Long (round-off to 3 decimals)

    When did Rio plan to meet Professor?
        Answer in DD-MM-YYYY_HH:MM:SS. (In original timezone)

    How many members did Rio gathered for the heist?
        Format: N (Integer)

    How many tasks did Rio created in planning the heist, and how many did he complete?
        Format: Created_Completed in Integers

    There is a secret code present in a document shared between Rio & Professor, can you find out what it is?
        Format: AB1234578PMFE

    We found a game installed on the device. When did Rio first open this game?
        Answer in DD-MM-YYYY_HH:MM:SS in UTC

Note:

    Wrap the flag around inctf{}
    Sample flag: inctf{01-01-2012_14:01:16_1.111_2.222_01-01-2012_14:01_1_0_0_AB1234578PMFE_01-01-2012_14:01:00}
    Flag is Case Sensitive
